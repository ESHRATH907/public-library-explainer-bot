# public-library-explainer-bot
Public Library Services Explainer Bot , This project builds a Generative AI chatbot that explains public library services.
The bot clarifies membership rules, borrowing limits, and digital resources.
Gemini Flash models ensure quick and friendly explanations.
A Streamlit web interface enables easy access.
System prompts restrict the bot to explanation-only behavior.
No account or transaction handling is allowed.
This project demonstrates safe GenAI use in public knowledge services.
It enhances user experience and reduces staff workload.
